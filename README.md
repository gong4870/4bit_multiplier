# 4bit_multiplier
4bit multiplier written in verilog

veilog를 이용해 multiplier를 구현해볼 예정이며, 내가 구현할 곱셈 방식은 Add-Shift multiplier이다.

내가 구현하려는 최종 형태는 Nexy-Board를 이용하여 입력 값을 SW로 받으면 최종 결과 값(10진수)을 seven-segment로 나오는 형태이다.
